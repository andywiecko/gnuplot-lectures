# gnuplot-lectures
gnuplot step-by-step examples

* install gnuplot: `sudo apt install gnuplot` (recommended version 5 or higher: https://sourceforge.net/projects/gnuplot/files/gnuplot/)

* plot in terminal with `gnuplot plot.plt`
